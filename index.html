Crystal Collector

Collect crystals below to match the number displayed!

A random number will be generated at the start of each game. The four crystals below each have their own unique numerical value. Click a crystal to add its value to your user score. The objective is to have your user score match the target score.

Be careful! If you go over the target score you loose! Once you match (or go over) the target score, the game will reset for another go. A new target score will be displayed and new values given to the crystals.
